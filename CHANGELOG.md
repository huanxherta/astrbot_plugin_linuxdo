# 更新日志 (CHANGELOG)

## [1.6.0] - 2026-04-16
### 新增
- **最新发布指令**: 新增 `/ld_fresh` 指令，通过 `order=created` 接口获取真正的新开话题，排除被回复顶起的旧帖。
### 优化
- **配置扩展**: 增加了对 `fresh_limit` 的支持，用户可独立设置新贴的获取数量。
- **描述微调**: 区分了“活跃讨论”和“最新发布”的概念。

## [1.5.0] - 2026-04-16
### 新增
- **自动翻页**: 核心抓取逻辑支持分页获取，可一次性拉取最多 100 条话题。

## [1.4.0] - 2026-04-16
### 变更
- **精简指令**: 移除了 `/ld` 搜索功能，规避社区 API 限制。

## [1.0.2] - 2026-04-16
### 修复
- **绕过屏蔽**: 引入 `curl_cffi` 解决 Cloudflare 403 拦截。
