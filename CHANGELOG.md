# 更新日志 (CHANGELOG)

## [1.0.9] - 2026-04-16
### 修复
- **架构对齐**: 按照 AstrBot 官方文档重构合并转发逻辑，使用标准 `Node` 列表。
- **稳定性**: 移除冗余组件，提高消息发送成功率。

## [1.0.8] - 2026-04-16
### 优化
- **轻量化折叠**: 尝试通过单节点转发规避 NapCat/LLOneBot 超时问题。

## [1.0.7] - 2026-04-16
### 变更
- **临时方案**: 为解决转发超时，临时切换为极简文本模式。

## [1.0.6] - 2026-04-16
### 修复
- **校验错误**: 修正 Node ID 的数据类型为整数 (Integer)，对齐 Pydantic 校验规范。

## [1.0.5] - 2026-04-16
### 优化
- **UI 升级**: 增加合并转发 (Forward) 功能，支持 15 条热帖折叠显示，防止刷屏。

## [1.0.2] - 2026-04-16
### 修复
- **绕过屏蔽**: 引入 `curl_cffi` 模拟 Chrome TLS 指纹，完美解决 Cloudflare 403 Forbidden 拦截。

## [1.0.1] - 2026-04-16
### 修复
- **Header 伪装**: 增加基础 User-Agent 伪装。

## [1.0.0] - 2026-04-16
### 新增
- **初始版本**: 支持 /ld_top 获取热帖和 /ld 搜索 LINUX DO 社区内容。
