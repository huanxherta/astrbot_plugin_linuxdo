# 更新日志 (CHANGELOG)

## [1.4.0] - 2026-04-16
### 变更
- **精简指令**: 移除了 `/ld` 搜索功能，以规避社区 API 的频繁 429 限制及改善稳定性。
### 优化
- **文档同步**: 更新了 README 和帮助信息，使其更加聚焦于热点与最新内容的获取。

## [1.3.7] - 2026-04-16
### 修复
- **WebUI 配置**: 在 `metadata.yaml` 中补充了 `config_schema` 声明，解决了旧版本 AstrBot 不显示配置界面的问题。

## [1.3.6] - 2026-04-16
### 修复
- **兼容性**: 彻底移除 `config_schema` 装饰器参数，确保在 AstrBot v4.22.1 及更早版本中正常加载。

## [1.2.1] - 2026-04-16
### 优化
- **单一气泡**: 强制使用单一 Node 封装所有内容，确保只发送一个合并转发气泡。

## [1.0.2] - 2026-04-16
### 修复
- **绕过屏蔽**: 引入 `curl_cffi` 完美解决 Cloudflare 403 拦截。

## [1.0.0] - 2026-04-16
### 新增
- **初始版本**: 支持获取热帖和社区内容。
