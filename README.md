# AstrBot LINUX DO 助手

获取 [LINUX DO](https://linux.do) 社区的实时动态。

## 🛠️ 使用指令

| 指令 | 说明 |
| :--- | :--- |
| `/ld_top` | **每日热榜**: 获取 24h 内最热门的话题。 |
| `/ld_new` | **活跃讨论**: 获取最近有新回复的话题。 |
| `/ld_time` | **最新发布**: 仅获取刚发布的新帖 (不含旧帖评论)。 |

## 🌟 核心特色

- **海量抓取**: 支持自动翻页，单次最高可获取 100+ 条话题。
- **智能分块**: 自动适配 QQ 字数限制，大批量发送不丢包。
- **完美折叠**: 合并转发气泡展示，整洁不刷屏。
- **WebUI 配置**: 支持在管理后台直接调整抓取数量和过滤规则。
- **稳定绕过**: 内置浏览器指纹伪装，有效规避 403 拦截。

## 📦 安装方法

1. 在 AstrBot 管理后台点击“从 URL 安装”。
2. 输入地址：`https://github.com/huanxherta/astrbot_plugin_linuxdo`
3. 重启机器人即可生效。
